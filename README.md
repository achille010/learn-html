# Welcome to Learn HTML!

This is a comprehensive, modern, and accessible guide to mastering HTML5. Whether you are a total beginner or an experienced dev looking for a refresher, this course has something for you.

## Curriculum Overview

-   **01-Foundations**: The basics of how the web works.
-   **02-Text and Content**: Formatting words and paragraphs.
-   **03-Links and Navigation**: Moving between pages.
-   **04-Images and Media**: Adding visuals and sound.
-   **05-Lists**: Organizing information.
-   **06-Tables**: Handling structured data.
-   **07-Forms**: Interactive user input.
-   **08-Semantic HTML**: Writing code with meaning.
-   **09-Head and Metadata**: Managing page settings.
-   **10-Accessibility**: Building for everyone.
-   **11-Advanced Topics**: Modern APIs and components.
-   **12-HTML and CSS Bridge**: How styling works.
-   **13-HTML and JavaScript Bridge**: How behavior works.
-   **14-Projects**: Real-world practice.

## How to use this guide
Each lesson contains an explanation, code examples, and "Pro Tips." We recommend following the modules in order, but feel free to jump to any topic you need!
