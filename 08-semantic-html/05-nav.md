# Nav

The `<nav>` tag is used specifically for sections of the page that contain navigation links.

## Usage
While you can have multiple `<nav>` elements on a page (e.g., one in the header and one in the footer), it should be reserved for **major** navigation blocks.

```html
<nav>
    <ul>
        <li><a href="/">Home</a></li>
        <li><a href="/products">Products</a></li>
        <li><a href="/faq">FAQ</a></li>
    </ul>
</nav>
```
