# Module 08: Semantic HTML

In this module, we move beyond simple structure and learn how to write HTML that carries **meaning**.

## What's in this module?

1.  **What is Semantic HTML?**: Writing code for computers and humans.
2.  **Header and Footer**: Defining the top and bottom of your page.
3.  **Main and Section**: Organizing the core content.
4.  **Article and Aside**: Handling independent content and sidebars.
5.  **Nav**: Defining navigation blocks.
6.  **Figure and Time**: Captions and machine-readable dates.
7.  **Mark, Abbr, Cite**: Inline semantic tags.
8.  **Div vs. Semantic Elements**: When to use each.
9.  **SEO Benefits of Semantic HTML**: Why Google loves it.
10. **Practice Exercises**: Test your knowledge of semantic HTML.
