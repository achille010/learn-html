# Main and Section

These tags are used to organize the core content of your page.

## `<main>`
The `<main>` tag should contain the primary content of the page. It should be **unique** to the document (only one per page). 
> [!TIP]
> Do not put site-wide sidebars or navigation inside `<main>`.

## `<section>`
The `<section>` tag is used for grouping related content together. Each section usually has its own heading.

```html
<main>
    <section>
        <h2>Latest News</h2>
        <p>Stay up to date with our blog.</p>
    </section>
    
    <section>
        <h2>Upcoming Events</h2>
        <p>Join us for our next meeting.</p>
    </section>
</main>
```
