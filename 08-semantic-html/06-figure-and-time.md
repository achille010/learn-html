# Figure and Time

These tags help you provide extra meaning to images and dates.

## `<figure>` and `<figcaption>`
As we learned earlier, these group an asset with its caption.

## `<time>`
The `<time>` tag is used to make dates and times machine-readable. This is great for search engines and calendar apps. Use the `datetime` attribute for the computer-friendly version.

```html
<p>This post was published on <time datetime="2024-03-01">March 1st, 2024</time>.</p>
```
