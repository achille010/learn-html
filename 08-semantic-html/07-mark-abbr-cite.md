# Mark, Abbr, Cite

These inline tags provide specific meaning to parts of your text.

## `<mark>`
Used to **highlight** or mark text for reference (usually rendered with a yellow background).

```html
<p>The <mark>deadline</mark> is tomorrow.</p>
```

## `<abbr>`
Used for **abbreviations**. Use the `title` attribute to show the full version when the user hovers.

```html
<p>Working with <abbr title="HyperText Markup Language">HTML</abbr> is fun!</p>
```

## `<cite>`
Used to reference a creative work (like a book, movie, or song title). It is usually rendered in *italics*.

```html
<p>I learned a lot from <cite>The HTML Bible</cite>.</p>
```
