# Document Structure

Every HTML page should follow a standard "boilerplate" structure.

## The Boilerplate

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Default Page Structure</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>
```

## Breakdown
-   `<!DOCTYPE html>`: Tells the browser this is an HTML5 document.
-   `<html lang="en">`: The root element, specifying the language.
-   `<head>`: Contains metadata (info about the page) like the title and character encoding.
-   `<body>`: Contains all the visible content.
