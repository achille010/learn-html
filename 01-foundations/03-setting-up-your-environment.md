# Setting up Your Environment

You don't need much to start writing HTML, but having the right tools makes it a lot easier.

## 1. Text Editor
While you *could* use Notepad, professional developers use specialized editors.
**Recommendation**: [VS Code (Visual Studio Code)](https://code.visualstudio.com/). It's free, powerful, and has great extensions for HTML.

## 2. Web Browser
You likely already have one! Chrome or Firefox are recommended for their excellent "Developer Tools."

## 3. Extensions (VS Code)
If you're using VS Code, install the **Live Server** extension. It allows you to see your changes instantly as you save your file.
