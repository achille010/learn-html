# Comments

**Comments** are notes left in the code for developers. They are not displayed in the browser.

## The Syntax
To write a comment in HTML, use this format:
`<!-- Everything in here is a comment -->`

```html
<p>Visible text.</p>
<!-- This is a hidden note for myself. -->
```

## Why use comments?
1.  **Explanation**: Describe what a complex section of code does.
2.  **Organization**: Separate different parts of your page.
3.  **Debugging**: Temporarily "hide" code to see if it fixes an issue.
