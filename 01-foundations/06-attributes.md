# Attributes

**Attributes** provide additional information about an element. They always appear in the opening tag.

## The Syntax
Attributes usually come in name/value pairs like: `name="value"`.

```html
<a href="https://google.com">Click me!</a>
```
In this example, `href` is the attribute name, and the URL is the value.

## Common Attributes
-   `id`: A unique identifier for an element.
-   `class`: A non-unique identifier used for styling multiple elements.
-   `src`: Specifies the path to an image or file.
-   `alt`: Alternative text for an image or element.
