# Your First HTML Page

Let's get our hands dirty and write some code!

## Step 1: Create a file
Open your text editor and create a new file named `index.html`. 
> [!TIP]
> Always use the `.html` extension so the browser knows how to read it.

## Step 2: Add content
Type the following into your file:

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>This is my first HTML page.</p>
</body>
</html>
```

## Step 3: Open in browser
Right-click your file and select "Open with..." and choose your browser. You should see "Hello World!" in big bold letters!
