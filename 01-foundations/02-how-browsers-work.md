# How Browsers Work

To be a great web developer, it's helpful to understand what happens "under the hood" when you open a website.

## The Process

1.  **Request**: You type a URL (like `google.com`) or click a link. Your browser sends a request to a server.
2.  **Response**: The server sends back files (HTML, CSS, JS).
3.  **Parsing**: The browser reads the HTML and builds the **DOM** (Document Object Model).
4.  **Rendering**: The browser takes the DOM and the CSS styles to paint the pixels on your screen.

## Popular Browsers
Common browsers include Chrome, Firefox, Safari, and Edge. While they all aim to follow the same standards, they sometimes render things slightly differently.
