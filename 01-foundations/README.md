# Module 01: Foundations

Welcome to the start of your HTML journey! In this module, we will cover the core concepts that form the backbone of the web.

## What's in this module?

1.  **What is HTML?**: Understanding the role of HyperText Markup Language.
2.  **How Browsers Work**: A high-level look at how your code becomes a website.
3.  **Setting up Your Environment**: Tools you need to start coding.
4.  **Your First HTML Page**: Writing and viewing your very first file.
5.  **Tags and Elements**: The building blocks of HTML.
6.  **Attributes**: Adding extra information to your elements.
7.  **Comments**: How to leave notes in your code.
8.  **Document Structure**: The standard boilerplate for every HTML page.
9.  **Practice Exercises**: Test your knowledge of foundations.
