# What is HTML?

**HTML** stands for **HyperText Markup Language**. It is the standard markup language for documents designed to be displayed in a web browser.

## Key Concepts

-   **HyperText**: Refers to the "links" that connect web pages to one another.
-   **Markup**: HTML uses "markup" (tags) to annotate text, images, and other content for display.
-   **Structure**: Think of HTML as the *skeleton* of a webpage. CSS is the *skin* (styling), and JavaScript is the *muscle* (behavior).

## A Brief History
HTML was created by **Tim Berners-Lee** in 1991. Since then, it has evolved through many versions, with **HTML5** being the current modern standard.
