# Practice Exercises: Foundations

Test your knowledge of the core concepts of HTML and the web.

## Quiz

1. What does HTML stand for?
   - A) HighText Markup Language
   - B) HyperText Markup Language
   - C) Hyperlink and Text Management Language
   - D) Home Tool Markup Language

2. Which tag is used to define the root of an HTML document?
   - A) `<root>`
   - B) `<body>`
   - C) `<html>`
   - D) `<head>`

3. Where should the `<title>` tag be placed?
   - A) Inside the `<body>`
   - B) Inside the `<head>`
   - C) Before the `<html>` tag
   - D) After the `</html>` tag

4. What is the purpose of the `<!DOCTYPE html>` declaration?
   - A) To display a header on the page
   - B) To tell the browser which version of HTML is being used
   - C) To link a CSS file
   - D) To create a comment

## Coding Challenge: The Basic Structure

Create a new file named `index.html` and write the complete boilerplate structure for an HTML5 document. 

Requirements:
- Include the `<!DOCTYPE>` declaration.
- Set the language to English.
- Add a `<title>` that says "My First Webpage".
- Add a comment inside the `<body>` that says "Hello World".

## Bonus Challenge

Research the `meta` tag for character encoding (`charset`). Why is it important to include `<meta charset="UTF-8">` in your `<head>`?
