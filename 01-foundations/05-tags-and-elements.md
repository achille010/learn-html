# Tags and Elements

HTML is made up of **elements**. Elements are the individual components of a page.

## The Anatomy of an Element
An element is usually composed of:
1.  **Opening Tag**: `<p>`
2.  **Content**: `This is a paragraph.`
3.  **Closing Tag**: `</p>`

Combining these gives us an element: `<p>This is a paragraph.</p>`

## Self-Closing Tags
Some elements don't have content or a closing tag. They are "self-closing."
-   `<br>`: Line break.
-   `<hr>`: Horizontal rule.
-   `<img>`: Image element.
