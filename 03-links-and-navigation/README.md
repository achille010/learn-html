# Module 03: Links and Navigation

Links are the essence of the web. This module explains how to connect your pages and navigate between them.

## What's in this module?

1.  **Anchor Tags**: The foundational `<a>` tag.
2.  **Relative vs. Absolute URLs**: Knowing where your links point.
3.  **Opening Links in a New Tab**: Using the `target` attribute.
4.  **Linking to Page Sections**: Creating "jump links" with IDs.
5.  **Email and Phone Links**: Using `mailto:` and `tel:`.
6.  **Download Links**: Forcing a file to download.
7.  **Navigation Menus**: Using `<nav>` and lists for menus.
