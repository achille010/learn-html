# Anchor Tags

The `<a>` tag (anchor tag) is used to create hyperlinks.

## Basic Syntax

```html
<a href="https://example.com">Visit Example.com</a>
```

-   **href**: The "hypertext reference" or destination of the link.
-   **Link Text**: The text between the tags that the user clicks.

## Linking to Your Own Pages
If you want to link to another file in the same folder:

```html
<a href="contact.html">Contact Us</a>
```
