# Email and Phone Links

You can create links that automatically open the user's email client or dial a phone number.

## Email Links (`mailto:`)

```html
<a href="mailto:support@example.com">Email Support</a>
```

## Phone Links (`tel:`)
These are especially useful for mobile users.

```html
<a href="tel:+15551234567">Call Us</a>
```

> [!NOTE]
> Ensure the phone number includes the country code for the best results on mobile devices.
