# Download Links

If you want to force a file to download rather than being opened in the browser, use the `download` attribute.

## Basic Usage

```html
<a href="resume.pdf" download>Download my Resume</a>
```

You can also specify a new name for the downloaded file:

```html
<a href="resume.pdf" download="Achille_Resume_2024">Download Resume</a>
```

> [!TIP]
> This attribute only works for files on the same origin (the same domain).
