# Practice Exercises: Links and Navigation

Practice creating different types of links and building a navigation structure.

## Quiz

1. Which attribute is required for an anchor (`<a>`) tag to function as a link?
   - A) `src`
   - B) `link`
   - C) `href`
   - D) `id`

2. What does `target="_blank"` do?
   - A) Opens the link in the same tab.
   - B) Opens the link in a new window or tab.
   - C) Clears the page before loading the link.
   - D) Disables the link.

3. How do you link to a section on the same page with the ID `contact`?
   - A) `<a href="contact">`
   - B) `<a href="#contact">`
   - C) `<a href=".contact">`
   - D) `<a href="page.html#contact">` (only from another page)

4. Which protocol is used to create a link that opens the user's email client?
   - A) `http:`
   - B) `mail:`
   - C) `mailto:`
   - D) `email:`

## Coding Challenge: Creating a Mini-Site

Create two HTML files: `index.html` and `about.html`.

In `index.html`:
- Create a navigation menu with links to both pages.
- Add an external link to `https://www.google.com` that opens in a new tab.
- Add an internal link to a "Contact" section at the bottom of the same page.

In `about.html`:
- Create the same navigation menu.
- Add a "back to home" link.
- Add a link to your email address.

## Bonus Challenge

Research the `rel="noopener noreferrer"` attribute. Why should you use it when using `target="_blank"`?
