# Ordered Lists

An ordered list is used for items that follow a specific sequence (like steps in a recipe). By default, items are numbered.

## Basic Syntax

```html
<ol>
    <li>Wake up</li>
    <li>Brush teeth</li>
    <li>Code HTML</li>
</ol>
```

## Useful Attributes
-   **type**: Change the numbering style (`1`, `A`, `a`, `I`, `i`).
-   **start**: Start numbering from a specific number.
-   **reversed**: Display the sequence in reverse order.

```html
<ol type="A" start="3">
    <li>Item C</li>
    <li>Item D</li>
</ol>
```
