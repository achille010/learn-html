# Practice Exercises: Lists

Get hands-on experience with unordered, ordered, nested, and description lists.

## Quiz

1. Which tag is used to create an unordered (bulleted) list?
   - A) `<ol>`
   - B) `<ul>`
   - C) `<li>`
   - D) `<dl>`

2. How do you define a list item within a list?
   - A) `<item>`
   - B) `<list>`
   - C) `<li>`
   - D) `<ul-item>`

3. Which list type is best for a glossary of terms?
   - A) Ordered List
   - B) Unordered List
   - C) Description List
   - D) Nested List

4. In a description list (`<dl>`), what does the `<dd>` tag represent?
   - A) Description Data
   - B) Description Definition (the description itself)
   - C) Description Detail
   - D) Description Divider

## Coding Challenge: The Multi-Part List

Create an HTML page that includes the following:
- An **ordered list** of your top 3 favorite movies.
- An **unordered list** of the ingredients for your favorite sandwich.
- A **nested list**: Create an unordered list of categories (e.g., "Fruits", "Vegetables"). Inside each category, add an unordered list of specific items.
- A **description list** for three tech terms (e.g., "Frontend", "Backend", "Fullstack") with their definitions.

## Bonus Challenge

Research the `type` and `reversed` attributes for the `<ol>` tag. How can you change the numbering to Roman numerals?
