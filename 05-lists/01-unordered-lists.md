# Unordered Lists

An unordered list is used for items that don't need to follow a specific sequence. By default, items are displayed with bullet points.

## Basic Syntax
An unordered list uses the `<ul>` tag, and each item inside it uses the `<li>` (List Item) tag.

```html
<ul>
    <li>Apples</li>
    <li>Bananas</li>
    <li>Oranges</li>
</ul>
```

## Styling Bullets
While you can change the bullet type in HTML using the `type` attribute (e.g., `<ul type="square">`), modern developers use **CSS** for this purpose.
