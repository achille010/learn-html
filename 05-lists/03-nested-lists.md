# Nested Lists

You can put one list inside another. This is called "nesting."

## Key Rule
The nested list must be **inside** a `<li>` tag of the parent list.

```html
<ul>
    <li>Fruits
        <ul>
            <li>Apples</li>
            <li>Bananas</li>
        </ul>
    </li>
    <li>Vegetables
        <ul>
            <li>Carrots</li>
            <li>Broccoli</li>
        </ul>
    </li>
</ul>
```

## Common Use Cases
-   **Navigation menus**: Creating sub-menus or drop-downs.
-   **Table of Contents**: Organizing complex documents into hierarchies.
