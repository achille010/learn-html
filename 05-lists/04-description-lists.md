# Description Lists

A description list is used to group terms and their descriptions (like a glossary).

## Tags Involved
-   `<dl>`: The whole list (Description List).
-   `<dt>`: The term (Description Term).
-   `<dd>`: The description (Description Definition).

## Basic Syntax

```html
<dl>
    <dt>HTML</dt>
    <dd>The standard markup language for creating web pages.</dd>
    <dt>CSS</dt>
    <dd>The style sheet language used to describe the presentation of a document.</dd>
</dl>
```
