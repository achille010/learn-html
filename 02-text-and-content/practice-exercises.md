# Practice Exercises: Text and Content

Master the art of formatting text and handling special characters in HTML.

## Quiz

1. Which heading tag represents the most important heading on a page?
   - A) `<h6>`
   - B) `<h4>`
   - C) `<h1>`
   - D) `<head>`

2. What is the difference between `<b>` and `<strong>`?
   - A) There is no difference in how they appear or are used.
   - B) `<b>` is for bold appearance, while `<strong>` indicates semantic importance (often for screen readers).
   - C) `<strong>` makes text larger than `<b>`.
   - D) `<b>` is deprecated and should never be used.

3. Which tag is used to create a line break without starting a new paragraph?
   - A) `<hr>`
   - B) `<break>`
   - C) `<lb>`
   - D) `<br>`

4. How would you display the less-than symbol (`<`) in an HTML paragraph?
   - A) Just type `<`
   - B) Use the entity `&lt;`
   - C) Use the entity `&less;`
   - D) Wrap it in `<code>`

## Coding Challenge: Formatting an Article

Create an HTML file that includes the following:
- An `<h1>` title for your article.
- An `<h2>` subtitle.
- A paragraph with some text in **bold** and some in *italics*.
- A `<blockquote>` containing a famous quote.
- Use an `<hr>` to separate the article from a footer.
- In the footer, include "Copyright &copy; 2024 Your Name".

## Bonus Challenge

Research the difference between the `<pre>` tag and the `<code>` tag. When would you use them together?
