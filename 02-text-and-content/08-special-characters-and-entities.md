# Special Characters and Entities

Some characters are "reserved" in HTML because they have special meanings (like `<` and `>`). To display these, you must use **entities**.

## Common Entities

-   `&lt;` displays as `<`
-   `&gt;` displays as `>`
-   `&amp;` displays as `&`
-   `&copy;` displays as `©`
-   `&nbsp;` displays as a **non-breaking space**.

## Example

```html
<p>To write a tag, use &lt;p&gt;.</p>
<p>Copyright &copy; 2024</p>
```
