# Bold, Italic, Underline

These tags are used to change the visual appearance of text within a paragraph.

## Bold
-   `<b>`: Physical bolding of text.
-   `<strong>`: Semantically important text (usually rendered as bold). **Recommended** for emphasis.

## Italic
-   `<i>`: Physical italicizing.
-   `<em>`: Emphasized text (usually rendered as italics). **Recommended** for meaning.

## Underline
-   `<u>`: Used to denote misspelled words or proper names (though rarely used in modern web design).

```html
<p>This is <strong>important</strong> and this is <em>emphasized</em>.</p>
```
