# Preformatted Text

When you want to display text exactly as written, including spaces and line breaks, use the `<pre>` tag.

## Why use `<pre>`?
It is most commonly used for displaying code snippets.

```html
<pre>
    function hello() {
      console.log("Hello World!");
    }
</pre>
```

## Differences from `<p>`
-   `<p>` ignores extra spaces and line breaks.
-   `<pre>` keeps them as-is and usually displays in a "monospaced" (fixed-width) font.
