# Paragraphs

The `<p>` tag is used to define a block of text.

## Basic Usage

```html
<p>This is a paragraph of text. Browsers automatically add some space before and after each paragraph.</p>
```

## Whitespace Handling
Browsers will ignore multiple spaces and line breaks inside your `<p>` tag. 

```html
<p>
    This text     will look
    like one line   to the browser.
</p>
```

To preserve whitespace, check out the lesson on [Preformatted Text](./06-preformatted-text.md).
