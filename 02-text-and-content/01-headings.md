# Headings

**Headings** are used to define the hierarchy of a document. HTML provides six levels of headings from `<h1>` to `<h6>`.

## Hierarchy

```html
<h1>Main Title (Page level)</h1>
<h2>Section Title</h2>
<h3>Sub-section Title</h3>
<h4>Minor Title</h4>
<h5>Tiny Title</h5>
<h6>Tiniest Title</h6>
```

## Best Practices
1.  **Use only one `<h1>` per page**: This is vital for SEO and clarity.
2.  **Don't skip levels**: Don't go from `<h1>` to `<h3>` without an `<h2>`. It confuses screen readers.
3.  **Choose headings for structure, not style**: Use CSS to change font size. Use HTML to define what the section is.
