# Superscript and Subscript

To display text slightly above or below the baseline, use these tags.

## Superscript (`<sup>`)
Used for exponents (e.g., 2<sup>2</sup>) or footnotes.

```html
<p>The speed of light is 2.99 &times; 10<sup>8</sup> m/s.</p>
```

## Subscript (`<sub>`)
Used for chemical formulas or mathematical variables.

```html
<p>Water is H<sub>2</sub>O.</p>
```
