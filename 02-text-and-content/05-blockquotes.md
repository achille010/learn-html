# Blockquotes

The `<blockquote>` element is used to indicate that the enclosed text is a quotation from another source.

## Basic Usage

```html
<blockquote>
    "Talk is cheap. Show me the code."
</blockquote>
<cite>- Linus Torvalds</cite>
```

## The `cite` Attribute
You can also specify a URL to the original source within the tag.

```html
<blockquote cite="https://en.wikiquote.org/wiki/Linus_Torvalds">
    "Talk is cheap. Show me the code."
</blockquote>
```
