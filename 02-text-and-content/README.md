# Module 02: Text and Content

Now that we know the basic structure, let's learn how to add and format text on our pages.

## What's in this module?

1.  **Headings**: Creating structure with `<h1>` through `<h6>`.
2.  **Paragraphs**: Organizing blocks of text with `<p>`.
3.  **Bold, Italic, Underline**: Styling text within a paragraph.
4.  **Line Breaks and Horizontal Rules**: Controlling spacing.
5.  **Blockquotes**: Quoting external sources.
6.  **Preformatted Text**: Displaying code and whitespace exactly as written.
7.  **Superscript and Subscript**: Dealing with footnotes and formulas.
8.  **Special Characters**: How to display symbols like `©` or `&`.
