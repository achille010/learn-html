# Line Breaks and Horizontal Rules

Sometimes you need to control the spacing on your page without creating a new paragraph.

## Line Break (`<br>`)
The `<br>` tag is a self-closing tag that forces the next piece of content onto a new line.

```html
<p>This is line one.<br>This is line two.</p>
```

## Horizontal Rule (`<hr>`)
The `<hr>` tag is used to create a horizontal line across the page. It's often used to separate thematic content.

```html
<h2>Section A</h2>
<p>Detail about Section A.</p>
<hr>
<h2>Section B</h2>
<p>Detail about Section B.</p>
```
