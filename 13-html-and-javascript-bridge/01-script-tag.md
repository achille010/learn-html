# Script Tag

The `<script>` tag is used to embed or reference executable code. This is usually JavaScript.

## Two Ways to Use it

### 1. External Script
This is the most common way to include JavaScript.

```html
<script src="my-code.js"></script>
```

### 2. Internal (Inline) Script
Used for small, one-off scripts.

```html
<script>
    console.log("Hello from inside the HTML!");
</script>
```

> [!TIP]
> Just like CSS, keeping your JavaScript in a separate file makes your code easier to manage.
