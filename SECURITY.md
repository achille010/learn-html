# Security Policy

## Scope

**learn-html** is a beginner learning repository containing static HTML examples and exercises. It does not include a backend, user authentication, or any production software — so traditional software versioning does not apply.

## Reporting a Vulnerability

If you discover a security issue in this repository (e.g. a malicious link, harmful script example, or sensitive data accidentally committed), please report it responsibly:

1. **Do not open a public issue** for security-related concerns.
2. Visit [@achille010's GitHub profile](https://github.com/achille010) and check the **Socials** section for contact links.
3. Reach out directly — I'm most active on **Discord**.
4. Describe the issue clearly, including the affected file(s) and potential impact.

You can expect an acknowledgement within **7 days**. Once confirmed, the issue will be patched and a fix committed as soon as possible.
