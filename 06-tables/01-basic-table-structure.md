# Basic Table Structure

An HTML table is built row by row.

## Tags Involved
-   `<table>`: The container for the entire table.
-   `<tr>`: A **Table Row**.
-   `<td>`: A **Table Data** cell (the actual content).

## Basic Example

```html
<table border="1">
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
    <tr>
        <td>Row 2, Cell 1</td>
        <td>Row 2, Cell 2</td>
    </tr>
</table>
```

> [!NOTE]
> The `border` attribute is used here just for visibility. In modern web design, you should use CSS for borders.
