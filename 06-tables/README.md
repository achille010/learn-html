# Module 06: Tables

Tables are used to display structured data in a grid format. While they were once used for layout, today they are strictly for data.

## What's in this module?

1.  **Basic Table Structure**: Using `<table>`, `<tr>`, and `<td>`.
2.  **Table Headers**: Using `<th>` for clarity.
3.  **Spanning Rows and Columns**: Using `rowspan` and `colspan`.
4.  **thead, tbody, tfoot**: Organizing large tables.
5.  **Accessible Tables**: Making tables readable for everyone.
6.  **When NOT to use Tables**: Avoid using tables for page layouts.
