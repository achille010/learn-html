# Input Types

The `<input>` tag is the most versatile form element. Changing the `type` attribute completely changes its behavior.

## Common Types

| Type | Purpose |
| :--- | :--- |
| `text` | Single line of text. |
| `password` | Masks the characters. |
| `email` | Checks for an @ symbol. |
| `number` | Restricted to numbers (shows a numeric keyboard on mobile). |
| `date` | Opens a calendar picker. |
| `color` | Opens a color picker. |

## Example

```html
<input type="text" placeholder="First Name">
<input type="email" placeholder="Email Address">
<input type="password" placeholder="Password">
```
