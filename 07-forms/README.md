# Module 07: Forms

Forms are how you collect information from users. From search bars to login screens, forms are everywhere.

## What's in this module?

1.  **Form Basics**: Using the `<form>` tag.
2.  **Input Types**: Exploring `text`, `password`, `email`, and more.
3.  **Labels**: Why the `<label>` tag is absolutely essential.
4.  **Textarea and Select**: Handling long text and dropdown menus.
5.  **Checkboxes and Radio Buttons**: Making choices.
6.  **Buttons**: Submitting and resetting forms.
7.  **Form Validation**: Ensuring users provide correct data.
8.  **Fieldset and Legend**: Grouping related form fields.
9.  **Datalist**: Providing autocomplete options.
10. **Form Accessibility**: Making forms usable for everyone.
