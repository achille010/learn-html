# Buttons

There are three main types of buttons you can use inside a form.

## Button Types
1.  **submit**: Sends the form data to the server. (Default)
2.  **reset**: Resets all fields in the form to their initial values.
3.  **button**: A generic button that does nothing by default (often used with JavaScript).

## Example

```html
<button type="submit">Submit My Data</button>
<button type="reset">Start Over</button>
<button type="button" onclick="alert('Hello!')">Click Me!</button>
```
