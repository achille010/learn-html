# Keyboard Navigation

Many users (not just those with disabilities) use only a keyboard to navigate the web.

## The Tab Key
The most important key is **Tab**, which moves the "focus" from one interactive element to the next.

## Tab Order
The "tab order" follows the order of elements in your HTML.
> [!CAUTION]
> If your visual layout (via CSS) doesn't match your HTML order, it will be very confusing for keyboard users.

## Skip Links
A great accessibility feature is providing a link that allows users to skip past the navigation and go straight to the main content.

```html
<a href="#main-content" class="skip-link">Skip to main content</a>
```
