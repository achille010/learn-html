# WCAG Overview

**WCAG** stands for **Web Content Accessibility Guidelines**. These are the official international standards for web accessibility, managed by the W3C.

## The POUR Principles
Every accessibility rule falls into one of these four categories:

1.  **P**erceivable: Information and UI components must be presented in ways users can sense (not just see).
2.  **O**perable: Users must be able to use the interface (e.g., navigation via keyboard).
3.  **U**nderstandable: The information and operation of the UI must be clear and easy to follow.
4.  **R**obust: Content must be compatible with current and future technologies (like assistive devices).

## Conformance Levels
-   **Level A**: The bare minimum. (Most sites fail this).
-   **Level AA**: The standard goal for many businesses and laws.
-   **Level AAA**: The highest level, reserved for specialized sites.
