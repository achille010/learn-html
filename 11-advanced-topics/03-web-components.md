# Web Components

Web Components allow you to create your own custom, reusable HTML tags!

## The Three Technologies
1.  **Custom Elements**: Defining new HTML tags.
2.  **Shadow DOM**: Keeping the component's styles and markup hidden from the rest of the page.
3.  **HTML Templates**: Defining the markup structure in advance.

## Example Concept

```html
<!-- Using a custom tag -->
<user-card name="John Doe" img="john.jpg"></user-card>
```

In your JavaScript, you would define what happens when this tag is used. This allows for powerful, component-based development without needing a framework like React or Vue!
