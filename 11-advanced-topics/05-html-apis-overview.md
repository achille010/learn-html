# HTML APIs Overview

Modern browsers offer much more than just rendering tags. They provide powerful Application Programming Interfaces (APIs) for deep interactivity.

## What's possible?
-   **Storage**: Saving user data across sessions.
-   **Hardware Access**: Camera, Microphone, and Location.
-   **File System**: Reading and writing files.
-   **Offline Support**: Making your site work without internet.

## Why use them?
Using these APIs allows you to build web applications that feel more like "native" desktop or mobile apps. We'll explore the most common ones in the following lessons.
