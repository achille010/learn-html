# Module 11: Advanced Topics

In this module, we explore the cutting-edge features of modern HTML, including graphics, custom elements, and powerful browser APIs.

## What's in this module?

1.  **Canvas Element**: Drawing 2D and 3D graphics with JavaScript.
2.  **SVG in Depth**: Animating and styling vector graphics.
3.  **Web Components**: Creating your own custom HTML tags.
4.  **Template and Slot**: Reusable HTML fragments.
5.  **HTML APIs Overview**: A quick look at what the browser can do.
6.  **Drag and Drop**: Enabling interactive UI elements.
7.  **Geolocation**: Accessing the user's location.
8.  **Local Storage Intro**: Saving data in the browser.
9.  **Service Workers Intro**: Enabling offline support.
10. **Progressive Enhancement**: Building for everyone, regardless of browser.
