# Global Attributes

These attributes can be applied to **any** HTML element.

-   `class`: Specifies one or more class names for an element (reusable).
-   `id`: Specifies a unique id for an element (unique).
-   `title`: Specifies extra information about an element (usually shown as a tooltip).
-   `style`: Specifies an inline CSS style for an element.
-   `lang`: Specifies the language of the element's content.
-   `hidden`: Specifies that an element is not relevant (hides it).
-   `tabindex`: Specifies the tabbing order of an element.
-   `data-*`: Used to store custom data private to the page or application.
