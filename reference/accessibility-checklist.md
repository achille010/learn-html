# Accessibility Checklist

Use this checklist to ensure your site is usable by everyone.

## General
-   [ ] Is the `lang` attribute set on the `<html>` tag?
-   [ ] Does every page have a unique, descriptive `<title>`?
-   [ ] Is the heading hierarchy logical (`h1` -> `h2` -> `h3`)?

## Images and Media
-   [ ] Do all `<img>` tags have meaningful `alt` text?
-   [ ] Do video and audio tags have fallback content?

## Forms
-   [ ] Does every input have a connected `<label>`?
-   [ ] Are error messages clear and accessible to screen readers?
-   [ ] Are required fields marked with `aria-required="true"`?

## Navigation
-   [ ] Can you navigate the entire site using only the **Tab** key?
-   [ ] Is there a visible focus indicator for the current element?
-   [ ] Is there a "Skip to Content" link?

## Design
-   [ ] Is the color contrast ratio at least 4.5:1 for normal text?
-   [ ] Is the text still readable when zoomed to 200%?
