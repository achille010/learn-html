# HTML5 API Lookup

A summary of the most common APIs available in modern browsers.

| API | Use Case | Support |
| :--- | :--- | :--- |
| **Geolocation** | Get user's latitude and longitude. | High (Requires Permission) |
| **Drag and Drop** | Make elements draggable. | High |
| **Web Storage** | Save data in the browser (local/session). | High |
| **Canvas** | Draw 2D/3D graphics with JS. | High |
| **Service Workers** | Offline support and background tasks. | High |
| **Web Workers** | Run JS in background threads. | High |
| **Web Components** | Create custom HTML tags. | High |
| **Web Socket** | Real-time two-way communication. | High |
