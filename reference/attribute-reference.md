# Attribute Reference

## Anchor Tags (`<a>`)
-   `href`: Target URL
-   `target`: `_blank` (new tab), `_self` (same tab)
-   `download`: Prompts file download

## Images (`<img>`)
-   `src`: Image path
-   `alt`: Alternative text (Required)
-   `width`, `height`: Visual dimensions

## Inputs (`<input>`)
-   `type`: `text`, `password`, `email`, `number`, `checkbox`, `radio`, etc.
-   `placeholder`: Hint text
-   `value`: Fixed or initial value
-   `required`: Boolean validation
-   `name`: Identifier for form submission

## Forms (`<form>`)
-   `action`: URL to send data to
-   `method`: `GET` or `POST`
