# Common Tags Cheat Sheet

## Structure
-   `<html>`: Root element
-   `<head>`: Metadata container
-   `<body>`: Visible content
-   `<header>`, `<footer>`, `<main>`, `<nav>`, `<section>`, `<article>`, `<aside>`: Semantic layout

## Text
-   `<h1>` - `<h6>`: Headings
-   `<p>`: Paragraph
-   `<ul>`, `<ol>`, `<li>`: Lists
-   `<strong>`, `<em>`: Bold and Italic (Semantic)
-   `<blockquote>`: Long quotes

## Links and Media
-   `<a>`: Link
-   `<img>`: Image
-   `<video>`, `<audio>`: Multimedia
-   `<canvas>`, `<svg>`: Graphics

## Forms
-   `<form>`: Container
-   `<input>`: Input field
-   `<label>`: Input label
-   `<button>`: Clickable button
-   `<select>`, `<option>`: Dropdown menu
