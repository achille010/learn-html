# Open Graph Tags

Open Graph tags (introduced by Facebook) allow you to control how your website's content is displayed when shared on social media.

## The Core Tags
These tags are placed in the `<head>` and start with `og:`.

```html
<meta property="og:title" content="The Ultimate HTML Guide">
<meta property="og:type" content="website">
<meta property="og:url" content="https://www.example.com/">
<meta property="og:image" content="https://www.example.com/thumbnail.jpg">
<meta property="og:description" content="Master HTML5 with this interactive tutorial.">
```

## Why use them?
Websites with rich previews (images and accurate descriptions) get much higher click-through rates on platforms like Facebook, LinkedIn, and Twitter.
