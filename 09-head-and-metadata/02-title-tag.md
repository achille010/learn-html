# Title Tag

The `<title>` element is required in all HTML documents. It defines the title of the document.

## Where it appears
1.  **Browser Tab**: It shows the text in the tab at the top of the browser.
2.  **Search Engines**: It is the main headline in search result listings.
3.  **Bookmarks**: It is the default name when a user saves your page.

## Best Practices
-   **Be unique**: Every page on your site should have a different title.
-   **Be descriptive**: Instead of "Home", use "Home | Achille's HTML Tutorial".
-   **Length**: Keep it under 60 characters so it doesn't get cut off in search results.
