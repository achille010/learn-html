# Practice Exercises: Head and Metadata

Test your understanding of the metadata and configurations that live in the HTML `<head>`.

## Quiz

1. Which tag in the `<head>` defines the title shown in the browser tab and search results?
   - A) `<meta title>`
   - B) `<header>`
   - C) `<title>`
   - D) `<h1>`

2. What is the purpose of `<meta name="viewport" content="width=device-width, initial-scale=1.0">`?
   - A) To set the background color of the site.
   - B) To ensure the website is responsive and displays correctly on mobile devices.
   - C) To link a video to the page.
   - D) To define the character encoding.

3. Which tag is used to link an external CSS stylesheet to your HTML document?
   - A) `<script>`
   - B) `<anchor>`
   - C) `<link>`
   - D) `<style>`

4. Where is the best place to put your `<script>` link for optimal page performance?
   - A) At the very top of the `<head>`.
   - B) Inside the `<title>` tag.
   - C) Just before the closing `</body>` tag.
   - D) Anywhere inside the `<html>` tag.

## Coding Challenge: Setting Up the "Brain"

Create an HTML file with a fully configured `<head>` section.

Requirements:
- Set the character encoding to UTF-8.
- Set the viewport for responsiveness.
- Add a `<title>`: "My Professional Portfolio".
- Add a `<meta>` description for your portfolio.
- Add Open Graph (`og:`) tags for a title and image.
- Link a CSS file named `styles.css`.
- Link a JavaScript file named `app.js` (place the script tag before the closing `</body>`).
- Add a link for a favicon.

## Bonus Challenge

Research the `robots` meta tag. How can you tell search engines NOT to index a specific page?
