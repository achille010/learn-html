# Meta Tags

The `<meta>` tag is used to specify metadata about the HTML document. It is always used inside the `<head>` element.

## Common Meta Tags

```html
<!-- Description for Search Engines -->
<meta name="description" content="Learn HTML from scratch with this comprehensive guide!">

<!-- Keywords (Less important today) -->
<meta name="keywords" content="HTML, Tutorial, Coding, Web Development">

<!-- Author of the page -->
<meta name="author" content="Achille">
```

## Why are they used?
Meta tags are used by browsers (how to display content), search engines (keywords/description), and other web services.
