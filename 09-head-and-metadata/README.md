While the `<body>` contains what users see, the `<head>` contains what the browser and search engines need to know.

4.  **Meta Charset and Viewport**: Ensuring correct text and mobile layouts.
5.  **Open Graph Tags**: How your site looks when shared on social media.
6.  **Favicon**: Adding that little icon in the browser tab.
7.  **Linking CSS**: Connecting your styles.
8.  **Linking JavaScript**: Adding behavior.
