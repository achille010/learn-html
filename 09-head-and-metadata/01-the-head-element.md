# The Head Element

The `<head>` element is a container for all the metadata (data about data) of an HTML document.

## Characteristics
-   It is placed between the `<html>` tag and the `<body>` tag.
-   Content inside the `<head>` is **not** displayed on the page.
-   It defines the document title, character set, styles, scripts, and other meta information.

## Primary Child Elements
-   `<title>`
-   `<style>`
-   `<meta>`
-   `<link>`
-   `<script>`
-   `<base>`
