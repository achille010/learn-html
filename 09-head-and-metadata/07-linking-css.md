# Linking CSS

To keep your HTML clean, you should always store your styles in a separate file and link them.

## The `<link>` Tag
This tag connects your external `.css` stylesheet to your document.

```html
<link rel="stylesheet" href="styles.css">
```

-   **rel**: Specifies the relationship between the document and the linked file. For CSS, it must be `stylesheet`.
-   **href**: The path to your CSS file.
