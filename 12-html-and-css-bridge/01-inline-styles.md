# Inline Styles

You can apply styles directly to an HTML element using the `style` attribute.

## Basic Syntax

```html
<p style="color: red; font-size: 20px;">This is a red paragraph.</p>
```

## Why avoid inline styles?
1.  **Maintenance**: It's hard to change styles across multiple pages if they are hard-coded in every tag.
2.  **Separation of Concerns**: HTML should be for content, and CSS should be for design.
3.  **Performance**: Inline styles can make your HTML files larger and slower to load.
