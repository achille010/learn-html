# Class and ID

The `class` and `id` attributes are the most common way to "hook" CSS to your HTML.

## Class (`class`)
-   **Reusable**: You can use the same class on many different elements.
-   **Multiple**: An element can have multiple classes (separated by spaces).

```html
<p class="text-large text-blue">Blue Large Text</p>
```

## ID (`id`)
-   **Unique**: An ID must be unique on the page. Only one element can have a specific ID.
-   **Priority**: IDs have higher "specificity" in CSS than classes.

```html
<nav id="main-navigation">...</nav>
```
