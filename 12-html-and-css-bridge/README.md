# Module 12: HTML and CSS Bridge

HTML provides the structure, but CSS provides the style. This module explains how they talk to each other.

## What's in this module?

1.  **Inline Styles**: The `style` attribute (and why to avoid it).
2.  **Class and ID**: The primary way to target elements for styling.
3.  **Data Attributes**: Storing custom data for CSS and JS.
4.  **How CSS Selects HTML**: A look at basic selectors.
5.  **Practice Exercises**: Test your knowledge of the HTML/CSS bridge.
