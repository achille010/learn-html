# Alt Text

The `alt` attribute (Alternative Text) is one of the most important attributes for images.

## Why is it important?
1.  **Accessibility**: Screen readers read the `alt` text to visually impaired users.
2.  **SEO**: Search engines use `alt` text to understand what the image is about.
3.  **Fallback**: If the image fails to load, the `alt` text is displayed instead.

## Tips for Good Alt Text
-   **Be descriptive**: Instead of `alt="dog"`, use `alt="Golden Retriever playing with a ball"`.
-   **Don't say "image of"**: Screen readers already announce that it's an image.
-   **Decorative images**: If an image is purely for decoration, use an empty `alt=""`.
