# img Tag Basics

The `<img>` tag is used to embed an image in a web page.

## Basic Syntax
The `<img>` tag is a **self-closing** tag. It requires the `src` attribute to work.

```html
<img src="path-to-your-image.jpg" alt="Description of image">
```

-   **src**: The URL or file path to the image.
-   **alt**: Alternative text (highly recommended).

## Attributes for Size
You can also specify the `width` and `height` in pixels directly in the HTML (though CSS is usually preferred).

```html
<img src="logo.png" alt="Company Logo" width="200" height="100">
```
