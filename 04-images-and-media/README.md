# Module 04: Images and Media

Websites aren't just text; they are rich with multimedia. This module covers how to handle images, audio, and video effectively.

## What's in this module?

1.  **img Tag Basics**: Displaying your first image.
2.  **Alt Text**: Why accessibility and SEO depend on it.
3.  **Image Formats Explained**: JPG vs. PNG vs. WebP vs. AVIF.
4.  **Responsive Images**: Using `srcset` for different screen sizes.
5.  **Figure and Figcaption**: Grouping images with captions.
6.  **Audio**: Embedding sound files.
7.  **Video**: Embedding local and remote videos.
8.  **YouTube and iFrames**: Embedding external content.
9.  **SVG Basics**: Scalable Vector Graphics.
