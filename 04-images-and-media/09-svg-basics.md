# SVG Basics

**SVG** stands for **Scalable Vector Graphics**. Unlike JPG or PNG, SVGs are made of mathematical paths, meaning they never lose quality when scaled up.

## Why use SVGs?
1.  **Small File Size**: Perfect for icons and logos.
2.  **Scalability**: Looks crisp on any screen resolution (Retina displays).
3.  **Styling**: You can change colors and shapes with CSS and JavaScript!

## Simple SVG Example

```html
<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>
```
