# Image Formats Explained

Not all image formats are created equal. Choosing the right one can make your site faster.

## Common Formats

| Format | Best For... | Supports Transparency? |
| :--- | :--- | :--- |
| **JPG** | Photographs with many colors. | No |
| **PNG** | Icons, logos, and images needing transparency. | Yes |
| **WebP** | Modern standard. Great quality at small sizes. | Yes |
| **AVIF** | Next-gen format. Even better compression than WebP. | Yes |

## Recommendation
Use **WebP** whenever possible for general use. Use **AVIF** for high-performance needs, and **PNG** only when you need lossless transparency.
