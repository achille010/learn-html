# Practice Exercises: Images and Media

Learn how to work with images, video, audio, and SVGs in HTML.

## Quiz

1. Which attribute provides a text description for an image if it fails to load?
   - A) `title`
   - B) `desc`
   - C) `alt`
   - D) `text`

2. Which image format is generally best for icons and logos that need to scale?
   - A) JPG
   - B) PNG
   - C) GIF
   - D) SVG

3. Which element is used to provide a caption for an image or video?
   - A) `<captionital>`
   - B) `<legend>`
   - C) `<figcaption>`
   - D) `<label>`

4. What is the purpose of the `srcset` attribute on an `<img>` tag?
   - A) To link to multiple different images at once.
   - B) To provide different image files for different screen sizes/resolutions.
   - C) To set a border for the image.
   - D) To crop the image.

## Coding Challenge: Building a Multimedia Gallery

Create an HTML page that includes:
- A local image with a proper `alt` attribute.
- A `<figure>` containing an image and a caption.
- An embedded YouTube video (using an `<iframe>`).
- A simple SVG (like a circle or square).
- An `<audio>` player with controls enabled.

## Bonus Challenge

Research the `loading="lazy"` attribute for the `<img>` tag. How does it help with website performance?
