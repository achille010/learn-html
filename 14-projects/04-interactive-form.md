# Project 4: Interactive Form

Build a complex "Contact Sales" or "User Registration" form.

## Requirements
-   **Fieldsets**: Group information like "Personal Info" and "Account Settings".
-   **Labels**: Ensure every input has a clearly connected `<label>`.
-   **Input Types**: Use a variety of types (`email`, `tel`, `date`, `checkbox`, `radio`).
-   **Datalist**: Provide autocomplete suggestions for a field.
-   **Validation**: Add `required`, `minlength`, and `pattern` attributes for basic validation.

## Semantic Checklist
-   Did you use `<legend>` inside each `<fieldset>`?
-   Is the "Submit" button a `<button type="submit">`?
-   Are you using the correct `type` for each input?
