# Practice Exercises: Projects

Reflect on your journey and test your ability to plan and structure complex HTML projects.

## Quiz

1. What is the first step in the "Progressive Enhancement" workflow for a new web project?
   - A) Writing the CSS for the layout.
   - B) Researching JavaScript libraries.
   - C) Building a solid, semantic HTML structure that works without styling.
   - D) Creating a logo in SVG.

2. Which project from this module is best for practicing with `rowspan` and `colspan`?
   - A) Personal Portfolio
   - B) Recipe Page (if including a nutritional facts table)
   - C) Simple Landing Page
   - D) Interactive Form

3. When building a complex form, which semantic tags should you use to group related information (like "Billing Address")?
   - A) `<div>` and `<span>`
   - B) `<section>` and `<h5>`
   - C) `<fieldset>` and `<legend>`
   - D) `<article>` and `<header>`

4. How do you ensure that your projects are accessible to users with visual impairments?
   - A) Use only bright colors.
   - B) Provide descriptive `alt` text for images and use semantic HTML tags that screen readers understand.
   - C) Make all text bold.
   - D) Disable all images.

## Coding Challenge: Project Planning

Pick a new project idea (e.g., a "Product Catalog" or a "Contact Directory").

**Task**: Create a "Master Structure" HTML file for this project.
Requirements:
- Plan out the `<head>` with necessary metadata (title, description, viewport).
- Use a `<header>` for the site name and main navigation.
- Use a `<main>` container for the core content.
- Use at least three different semantic block elements (e.g., `<article>`, `<section>`, `<aside>`).
- Include a `<table>` for some structured data.
- Include a `<form>` for some user interaction.
- Link to its own CSS and JS files using the `defer` attribute.
