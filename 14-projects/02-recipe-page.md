# Project 2: Recipe Page

Build a detailed page for your favorite food recipe.

## Requirements
-   **Title and Image**: Use `<h1>` and a high-quality `<img alt="...">`.
-   **Description**: A short introduction to the dish.
-   **Ingredients**: Use an unordered list (`<ul>`).
-   **Instructions**: Use an ordered list (`<ol>`).
-   **Cook Time**: Use the `<time>` tag to show how long it takes.
-   **Notes**: Use a `<blockquote>` for any tips or variations.

## Semantic Checklist
-   Did you use `<figure>` and `<figcaption>` for the main image?
-   Did you use `<strong>` or `<em>` for emphasis?
-   Is the document structure logical?
