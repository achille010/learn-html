# Project 5: Accessible Article

Create a long-form news or blog article with high accessibility.

## Requirements
-   **Main Heading**: A clear `<h1>` for the page title.
-   **Metadata**: Include the author's name and a `<time>` tag for the publish date.
-   **Semantic Content**: Use `<article>` for the main story and `<aside>` for a sidebar or pull-quote.
-   **Media**: Include an image with detailed `alt` text and a caption using `<figure>`.
-   **Navigation**: Add a "Table of Contents" using internal links (`#id`).
-   **Footer**: Include references or an "About the Source" section.

## Semantic Checklist
-   Did you use `aria-label` or `aria-describedby` when necessary?
-   Is the tab order logical for a keyboard-only user?
-   Are you using `<mark>` to highlight key points?
